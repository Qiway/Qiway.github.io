Win7/Win10下80端口被占用

发表于 2017-02-22

根据最近遇到的情况总结，应该是升级Win10后常见的问题。在网上看到一篇更为详细的博客

有些说是IIS服务占用的，可以在管理设置。也可以直接设置开机不启动这个服务。
具体参考博客:空の心

点击“开始”——>“管理工具”——>“管理您的服务器”，单机“添加或删除角色”，等检测程序运行结束后，发现服务器角色里面IIS服务器这一项显示是开启的，于是将其关闭。
目前状态不能操作服务。既然不能直接关闭，那么至少让它在服务器启动时不自动启动就好了。所以就用到了sc config http start= disabled这个命令。
