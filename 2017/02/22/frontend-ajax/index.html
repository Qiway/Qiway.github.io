发表于 2017-02-22

一、什么是AJAX

  1. AJAX即“Asynchronous JavaScript and XML”（异步的JavaScript与XML技术），指的是一套综合了多项技术的浏览器端网页开发技术。它可以实现Web页面的局部更新，而不需要重新加载整个页面。满足数据快速传输的同时，也带来更好的用户体验。
  2. AJAX最重要的两项性能。
    a、无需重新装载整个页面便能向服务器发送请求。
    b、对XML文档的解析和处理。

二、使用AJAX基本流程

  1. 获取Web页面所需数据。
  2. 通过AJAX发送数据到指定URL。
  3. 后台脚本需求接受到的请求，返回给页面。
  4. 页面根据请求结果，更新局部信息。
