PHP笔记——正则表达式

发表于 2017-02-22 更新于 2019-11-18 分类于 PHP

简述

PHP长期支持两种正则表达式实现，分别是Perl正则表达式和POSIX正则表达式。POSIX在5.3.0版本中已经被废弃。Perl是目前PHP唯一正式支持的。

正式表达式可以根据定义的语法规则描述或匹配数据。正则表达式本身只是字符串模式，用于配置某些文本。

通配符 . 匹配换号符 \n 之外的符号

  • .at 可匹配 mat、cat、#at

范围集

  • [a-z] 匹配a到z的字符
  • [^a-z] 匹配非a到z的字符

量词

  • 匹配0次以上的次数 *
  • 匹配1次以上的次数 +
  • 配置0次或1次 ？

子表达式

  • (very )*large 可匹配 large 、very large、very very large

子表达式计数

  • (very ){1,3} 可匹配 very 、very very 、very very very

定位

  • 脱字符号^ 表示字符串开头
  • $ 表示字符串结尾

分支

  • cat|dog|bird 匹配 cat或者dog或者bird
