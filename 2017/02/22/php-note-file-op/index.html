PHP笔记——文件操作

发表于 2017-02-22 更新于 2019-11-18 分类于 PHP

##文件处理

文件打开：fopen( );

写文件：fwrite( $fp, $outputstring) ;（fputs()是fwrite()的别名函数）;

关闭文件 fclose( $fp );

何时读完文件 while( !feof( $fp ) );

读一行文件 fgets( $fp, 999 )，遇见换号或者读了998B则结束

过滤PHP和HTML标签 fgetss( $fp );

读取一行，定界符分割成数组 array fgetcsv( $fp, $max_line_length, $delimiter );

读取整个文件 readfile( $file_path, $include_path, $remote_protocol ); fpassthru( $fp ); $filearray = file( $file_path );

读取一个字符 fgetc( $fp );

读取任意长度 fread( $fp, $length );

查看文件是否存在 file_exists( $filename );

确定文件大小 filesize( $filename );

删除一个文件 unlink( $filename );

文件中定位：复位 rewind( $fp ); 跳到指定位置 fseek( $fp, 0, SEEK_SET ); 返回指针位置 ftell($fp);

文件锁定 flock( $fp, LOCK_EX ); （注意：使用该函数，必须将其添加到所有使用文件的脚本中）
