PHP笔记——面向对象

发表于 2017-02-22 更新于 2019-11-18 分类于 PHP

OOP的3个基本概念：

  1. 封装：通过接口将用户与实际应用程序内部的实现原理进行分离。
  2. 继承：同属于一个超集的不同分类，通过继承获得超集的性质和行为（属性和方法）。
  3. 多态：指OOP能够根据使用类的上下文来重新定义或改变类的性质和行为。

属性

1. 作用域描述符：private、public、protected、final、static

  • public：公共属性可以由相应的对象直接操作和访问。
  • private：私有属性只能在定义属性的类的内部被访问。
  • protected：类本身和继承的子类都可以访问和操作保护属性。
  • final：阻止在子类中覆盖该属性。

方法

1. 方法作用域：public、private、protected、abstract、final、static

  • public：公共方法可以在任何位置任何时间访问。
  • private：private的方法只能在类的内部访问。
  • protected：标记为protected的类方法只能在类和子类中使用。
  • abstract：抽象方法
  • final：final的方法可以防止被子类覆盖。

2. 类型提示

类型提示可以确保传递给方法的对象确实是所期望的类的成员。
